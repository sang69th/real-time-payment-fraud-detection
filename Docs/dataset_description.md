# Dataset Description

This project uses the Credit Card Fraud Detection dataset.

Dataset characteristics:
- transactions made by European cardholders
- highly imbalanced fraud dataset
- anonymized features V1 to V28
- additional columns:
  - Time
  - Amount
  - Class

Target column:
- Class
  - 1 = fraud
  - 0 = non-fraud

Project challenge:
- fraud cases are very rare compared to normal transactions
- this makes fraud detection difficult and requires careful evaluation
