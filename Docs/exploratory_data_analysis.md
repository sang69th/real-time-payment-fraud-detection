# Exploratory Data Analysis

Dataset contains approximately 9849 transactions.

Fraud distribution:
- fraud transactions: very small percentage
- highly imbalanced dataset

Key findings:

1. Fraud transactions are extremely rare.
2. Most transactions have small amounts.
3. Transaction amount distribution is highly skewed.
4. Fraud detection will require careful model evaluation due to imbalance.
